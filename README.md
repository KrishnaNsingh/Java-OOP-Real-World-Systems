# Java OOP Real-World Systems

This repository contains real-world system implementations built using **Object-Oriented Programming (OOP)** concepts in Java.

The goal of this project is to demonstrate how OOP principles are applied in designing modular and scalable real-life applications.

---

## Systems Included

### 1. ATM System

Simulates basic ATM operations.

**Features:**

* PIN verification
* Balance Check
* Deposit Money
* Withdraw Money
* Account Lock after 3 wrong attempts

**OOP Concepts Used:**

* Abstraction (Insert class)
* Inheritance (Deposit, Withdraw, Check_Balance)
* Encapsulation (Amount class)

---

### 2. Online Shopping Cart

Simulates an online shopping experience.

**Features:**

* Product Listing
* Add to Cart
* Buy Now
* View Cart
* Payment Options (Card / Cash on Delivery)

**OOP Concepts Used:**

* Class & Objects (Product)
* Encapsulation (Cart management)
* Method Overriding (toString)

---

### 3. College Management System

Simulates a college administration system.

**Modules Included:**

* Student Admission
* Fee Payment System
* Attendance Management
* Timetable Management
* Faculty Communication

**Features:**

* Student Enrollment
* Fee Payment (UPI / Cheque / Cash)
* Attendance Tracking
* Exam Eligibility Check
* Timetable Creation & Update
* Parent Communication

**OOP Concepts Used:**

* Composition (Faculty uses Attendance, Timetable, Communication)
* Encapsulation (Student data)
* Abstraction (Functional separation)
* Modular Design
